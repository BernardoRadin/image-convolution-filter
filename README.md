## About the Project

This project applies **convolution filters to images**, allowing image processing through custom matrices or predefined filters.
Users can upload an image, configure a convolution matrix, and apply effects directly through the interface.

The project was developed for educational purposes, focusing on concepts related to **digital image processing** and **convolution operations**.

## Instructions

1. Choose the **matrix size**.
2. Enter a **convolution filter** in the fields available on the screen or select one from the **"Ready Filters"** menu.
3. Select an image by clicking the **"Select Image"** button.
4. Click the **"Submit"** button and wait for the program to apply the filter. The processed image will be displayed on the screen.
5. Save the image by right-clicking and selecting **"Save image as..."**.

## What is an Image Convolution Filter?

Image convolution is a technique used in **digital image processing** to modify an image by applying a mathematical operation between the image pixels and a matrix, commonly called a **kernel** or **filter**.

Each pixel in the image is recalculated based on the surrounding pixels and the values present in the convolution matrix. This process makes it possible to create different visual effects and image enhancements.

Common examples of convolution filters include:

* **Blur** — smooths the image and reduces details.
* **Sharpen** — enhances edges and image details.
* **Edge Detection** — highlights contours and object boundaries.
* **Emboss** — creates a 3D relief-like effect.
* **Gaussian Blur** — applies smoother blur for noise reduction.

Convolution is widely used in areas such as **computer vision**, **medical imaging**, **artificial intelligence**, and **photo editing software**.
